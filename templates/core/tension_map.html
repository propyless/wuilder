ℹ

Readings — Left and right columns follow the same numbering as the Spokes wheel map. Values are TM-1 deflection units; we convert to kgf using the chart you pick.

Variance — Each spoke is compared to a reference tension. The limit is ± this percentage from that reference.

Other side as % of reference — At 100%, references are Park-style: each side vs its own average. Any other value applies a side ratio: the non-reference side’s target is that percent of the reference side’s average (e.g. 84% for a typical disc rear).

Map — Spoke color shows how far a spoke is from its reference; rim dots T / L mean tighten or loosen when outside the band. Arrows by kgf show the same.
