# Park TM-1 chart data

`tm1_charts.json` lists TM-1 **deflection reading → tension (kgf)** pairs per spoke type, aligned with Park Tool’s published **TM-1 Conversion Table** PDF:

- https://parktool.com/assets/doc/product/TM-1_conv-table.pdf

Cross-check values against the current Park document or the Wheel Tension App before relying on them for critical wheels. The TM-1 is a comparative tool, not a calibrated laboratory instrument.
